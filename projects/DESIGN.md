# DESIGN.md

## 气质与意象
深渊废土。声波可视化。末日的寂静与共振。晨光透过裂缝洒在废弃的声学实验室，空气中漂浮着肉眼可见的紫色声波涟漪。一切都在低频震动。

## 配色方案
- 主背景：深渊黑 `#0a0a0f`（废弃实验室的黑暗深处）
- 声波紫：`#8b5cf6`（旧日回音的可视化频率）
- 警告红：`#ef4444`（污染度攀升的危险信号）
- 护甲蓝：`#3b82f6`（调音师的声学护盾）
- 净化绿：`#22c55e`（频率校准后的安宁）
- 声爆紫：`#a855f7`（畸变体的侵蚀印记）
- 卡面深色：`#13131a`（卡牌底色）
- 金色高光：`#eab308`（能力牌/稀有牌标识）
- 文字主色：`#e2e8f0`
- 文字次要：`#94a3b8`

## 字体排版
- 中文：思源黑体 / Noto Sans SC（Google Fonts .cn 域）
- 英文/数字：Rajdhani（科技感、声学仪表风格）
- 标题使用 Rajdhani 粗体 + 大号
- 正文使用 Noto Sans SC 常规
- 数值/伤害数字使用 Rajdhani + 等宽风格

## 视觉策略
- 污染刻度尺采用渐变色轨道（绿→黄→橙→红→深紫），配合脉冲呼吸动画
- 卡牌使用暗色底+霓虹边框风格，攻击牌红边、技能牌蓝边、能力牌金边
- 背景使用微弱的声波脉冲动画（CSS 渐变 + animation）
- 掷骰结果用弹跳动画展示
- 伤害数字飞出动画

## 动效与交互
- 声波脉冲呼吸：slow ease-in-out，2-3s 循环
- 卡牌悬浮：translateY -4px + 微弱紫色发光
- 污染刻度攀升：刻度条平滑增长 + 边缘红光渐显
- 掷骰：弹跳缓动 cubic-bezier(0.34, 1.56, 0.64, 1)
- 页面切换：fade + 微弱 scale

## 设计禁忌
- 禁止明快糖果色
- 禁止圆角超过 12px
- 禁止可爱风格插画
- 禁止大面积纯白背景
- 禁止使用默认蓝色作为主色
- 禁止扁平无层次的纯色卡片
